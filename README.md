# Keylogger Detector & Analyzer

An interactive security defense utility developed as part of the **IBM 15-Days Cyber Security Internship**. This project focuses on identifying unauthorized keyboard hooking mechanisms and suspicious background behaviors often associated with Keyloggers.

## 🚀 Features
- **Heuristic Scanning Simulation**: Scans active API endpoints like `SetWindowsHookEx`.
- **Clean UI Dashboard**: Interactive frontend built with HTML and CSS to trigger scans.
- **Security Logs**: Simulates defensive reporting structures used by Security Operations Centers (SOC).

## 🛠️ Tech Stack
- **Front-end**: HTML5, CSS3, JavaScript
- **Back-end**: Python 3

## 📁 File Structure
- `index.html`: The main dashboard for user interaction.
- `app.py`: Core detection logic simulation script.
